## Installation

* Installer les dépendances : ``npm install``
* Installer : ``npm install axios --save``
* Démarrer le serveur : ``npm start``
* Ouvrir dans votre navigateur : ``localhost::3000``

## Storybook

* Installer : ``npx sb init``
* Démarrer le serveur : ``npm run storybook``
